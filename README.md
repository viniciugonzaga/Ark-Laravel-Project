<h1 align="center">ARK RPG SYSTEM</h1>

<p align="center">
Sistema de RPG web inspirado em sobrevivência, evolução e narrativa dinâmica.
</p>

<hr>

<h2>🧬 Funcionalidades</h2>

<ul>
<li>Criação completa de fichas de personagem</li>
<li>Sistema de atributos (FOR, AGI, INT, VIG, SET)</li>
<li>Mutações, bônus e poderes</li>
<li>Rolagem de dados modular (D4 até D100)</li>
<li>Sistema de eventos aleatórios</li>
<li>Histórico de rolagens por personagem</li>
<li>Geração de PDF das fichas</li>
<li>Interface estilo ARK futurista</li>
</ul>

<hr>

<h2>⚙️ Tecnologias</h2>

<ul>
<li>Laravel (Backend)</li>
<li>Blade (Frontend)</li>
<li>TailwindCSS</li>
<li>JavaScript</li>
<li>MySQL / PostgreSQL</li>
</ul>

<hr>

<h2>🚀 Como rodar localmente</h2>

<pre>
git clone SEU_REPOSITORIO
cd projeto

composer install
cp .env.example .env
php artisan key:generate

php artisan migrate
php artisan serve
</pre>
