# Ark-Laravel-Project
Sistema web de RPG inspirado em ARK, focado em sobrevivência, evolução e narrativa dinâmica. O projeto permite criação de fichas completas de personagens, sistema de atributos, mutações, bônus, rolagem de dados avançada e eventos aleatórios. Inclui geração de PDF das fichas e interface interativa estilo terminal futurista.
